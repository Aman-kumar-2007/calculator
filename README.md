# 🧮 Calculator Web App

## 📌 Overview

This is a simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

The project is designed to strengthen core frontend development skills and understand DOM manipulation and event handling in JavaScript.

---

## 🚀 Features

* Basic arithmetic operations ➕ ➖ ✖️ ➗
* Clean and user-friendly UI
* Responsive design (works on mobile & desktop)
* Real-time input display
* Error handling (e.g., division by zero)

---

## 🛠️ Tech Stack

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

---

## 📂 Project Structure

```
calculator-app/
│
├── index.html      # Main HTML file
├── style.css       # Styling file
├── script.js       # JavaScript logic
└── README.md       # Project documentation
```

---

## ⚙️ How It Works

1. User clicks buttons on the calculator UI
2. Input is displayed on the screen
3. JavaScript handles operations and updates results
4. `=` button evaluates the expression
